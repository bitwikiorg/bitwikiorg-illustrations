# Example Prompts

## Plan an illustration set

```text
Use $bitwiki-illustrations. Do not generate images yet.
Analyze the following article and produce a six-image shot list.
For each image provide: placement, cognitive anchor, composition pattern,
BITwiki components, house style, geometry mapping, required labels, and alt-text intent.

<article>
```

## Generate an Intelligence Factory overview

```text
Use $bitwiki-illustrations to create one 16:9 editorial illustration explaining
The Intelligence Factory: BIThub as the live workcell layer, BITwiki as durable semantic memory,
and BITCORE as the recursive reasoning engine. Show the canonical lifecycle and compounding reuse.
Use the Intelligence Codex house style. Avoid slide formatting and long embedded text.
```

## Generate a BIThub workcell illustration

```text
Use $bitwiki-illustrations to show a BIThub topic becoming a bounded workcell.
Depict humans and agents collaborating inside an open cube with visible context,
tools, assignments, and artifact output. Use the modern technical monochrome style
with a concentrated spectral flow marking active work.
```

## Generate a BITwiki memory illustration

```text
Use $bitwiki-illustrations to explain BITwiki as long-term semantic memory.
Use a Semantic Lattice composition with provenance, ontology, linked concepts,
versioned artifacts, and a codex anchor. Render in the Intelligence Codex style.
No paragraphs inside the image.
```

## Generate a BITCORE illustration

```text
Use $bitwiki-illustrations to explain BITCORE as an orthogonal agentic reasoning engine.
Show a bounded cubic human system interfacing with an octagram-hypercube manifold.
Represent recursive reasoning, tool orchestration, memory-aware processing,
and multi-agent coordination. Use spectral color only on active reasoning paths.
```

## Generate the lifecycle

```text
Use $bitwiki-illustrations to create a six-stage Lifecycle Conveyor:
Task enters BIThub → Topic becomes a workcell → Agents / Constructs / COREs process the task →
Useful output becomes an Artifact → Stable knowledge moves into BITwiki →
Future humans and agents reuse it. Include a return path showing that useful intelligence compounds.
```

## Edit or clean an existing image

```text
Use $bitwiki-illustrations to edit this image.
Remove generated gibberish, watermarks, and incorrect labels.
Preserve the main composition, BITwiki geometry, monochrome base, and spectral intelligence flow.
Leave clean label zones for final typography.
```
