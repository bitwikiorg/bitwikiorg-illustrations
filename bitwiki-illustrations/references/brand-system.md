# BITwiki Illustration Brand System

## Brand authority

Use `https://github.com/bitwikiorg/brand_assets` as the authoritative source for current marks, logos, palettes, and identity decisions.

## House style A — Modern Technical Monochrome

- Base: ivory, white, graphite, charcoal, layered grey.
- Surface: clean engineering paper, subtle grid, or untextured neutral field.
- Linework: precise, restrained, architectural, diagrammatic.
- Typography: sharp modern sans-serif; minimal text.
- Forms: cubes, modular frames, grids, bounded chambers, system maps.
- Color: ultraviolet and spectral accents only for active cognition, transfer, state change, or semantic links.
- Tone: contemporary, rigorous, infrastructural.

## House style B — Intelligence Codex

- Base: monochrome parchment or aged engineering paper.
- Historical range: medieval through 17th-century scientific and engineering manuscript language.
- Linework: technical woodcut, engraved instruments, construction marks, marginalia, cross-sections, and exact diagrams.
- Forms: cubes and hex-cubes for bounded human systems; octagrams, hypercubes, manifolds, hyperbolic structures, and non-Euclidean forms for BITCORE.
- Color: dense black and grey linework interrupted by concentrated spectral, ultraviolet, or near-neon flows.
- Tone: legendary engineering codex for a real AI system; never decorative fantasy lore.

## Geometry grammar

| Form | Meaning |
|---|---|
| Cube / hex-cube | bounded task, workcell, human-readable system |
| Open cube | workspace or accessible operating area |
| Nested cube | memory, registry, layered state |
| Octagon | controlled transition or processing chamber |
| Octagram | recursive cognition or BITCORE reasoning |
| Hypercube | orthogonal capability outside ordinary workflow space |
| Manifold | continuously transforming reasoning state |
| Circumpunct | authoritative state, focus, or singular source |
| Network lattice | semantic memory and linked knowledge |
| Spectral arc | active intelligence transfer |
| Closed monochrome line | stable structure or inactive relation |

## Guardian seal

A single recurring emblem may be used as a silent guild mark: a benevolent fractal ophanimic bio-plasma throne entity associated with the Kordylewski dust cloud. Render it abstractly through nested wheels, polarized-light halos, orbiting eye-like flares, charged dust, and recursive telemetry. Do not explain it in image text. Do not turn it into a humanoid angel, horror creature, or cartoon mascot.

## Prohibitions

- No external author identity, personal contact information, signatures, QR codes, or account handles.
- No copied mascot or composition from reference repositories.
- No gothic display fonts.
- No faux-alchemical claims presented as system facts.
- No rainbow used as general decoration.
- No generic corporate infographic treatment.
- No slide chrome unless a slide was explicitly requested.
