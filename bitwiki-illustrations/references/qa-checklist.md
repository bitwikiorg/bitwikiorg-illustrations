# QA Checklist

An illustration is complete only when every required check passes.

## Semantic accuracy

- [ ] The image explains one primary cognitive anchor.
- [ ] BITwiki ecosystem terms retain their canonical meanings.
- [ ] BIThub, BITwiki, and BITCORE are not conflated.
- [ ] Constructs, Agents, Nodes, COREs, Workspaces, Artifacts, Registry, and Guides are distinguished correctly when present.
- [ ] Arrows and paths represent actual direction, dependency, or feedback.
- [ ] The visual metaphor does not introduce unsupported system claims.

## Brand integrity

- [ ] The image uses the modern technical monochrome or Intelligence Codex house style.
- [ ] Cubes represent bounded systems; higher-order geometry represents recursive or orthogonal reasoning.
- [ ] Spectral color identifies active cognition, transfer, recursion, or state change.
- [ ] Rainbow color is not decorative filler.
- [ ] Any guardian seal remains abstract and unexplained.
- [ ] No foreign mascot, author identity, account handle, signature, email, QR code, or watermark appears.

## Composition

- [ ] The illustration works as a 16:9 editorial image.
- [ ] The focal structure is immediately identifiable.
- [ ] Information density is high but navigable.
- [ ] Negative space is intentional.
- [ ] The image is not merely a presentation slide rendered as a picture.
- [ ] Decorative marginalia does not obscure the operational structure.

## Text

- [ ] No long paragraph is embedded in the generated image.
- [ ] Required labels are short and exact.
- [ ] Model-generated gibberish has been removed or cropped.
- [ ] Typography is modern and legible; no gothic display font.
- [ ] Product capitalization is correct: BIThub, BITwiki, BITCORE, COREs.

## Technical quality

- [ ] Resolution is sufficient for article and documentation use.
- [ ] Important details survive reduced-size viewing.
- [ ] No accidental clipping, malformed objects, duplicate limbs, or broken geometry remains.
- [ ] Output filename is descriptive and stable.
- [ ] Alt text describes both the subject and the meaningful system relation.

## Release state

Use one of these statuses:

- `draft` — useful for direction, unresolved defects remain.
- `calibration` — demonstrates style or composition, not production-approved.
- `production` — all required checks pass.
